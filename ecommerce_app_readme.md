# Ecommerce Mobile App

[GitHub Repository](https://github.com/Karim-Deghidy/Ecommerce-Mobile-App)

## Overview
This project is a fully functional **mobile e-commerce application** built using **React Native** for both iOS and Android platforms. The app integrates modern features including user authentication, product catalog, shopping cart, checkout system, and in-app purchase handling. It is designed to provide a seamless shopping experience on mobile devices.

## Features
- User authentication (sign up, login, password recovery)
- Browse products with search and category filtering
- Product detail pages with images, descriptions, and pricing
- Shopping cart management
- Checkout with native in-app purchase support
- Order history and user profile management
- Responsive UI optimized for mobile devices
- Git version control with clear commit history

## Tech Stack

## Installation
1. Clone the repository:
```bash
git clone https://github.com/Karim-Deghidy/Ecommerce-Mobile-App.git
```
2. Navigate to the project directory:
```bash
cd Ecommerce-Mobile-App
```
3. Install dependencies:
```bash
npm install
```
4. Run the app (iOS):
```bash
npx react-native run-ios
```
5. Run the app (Android):
```bash
npx react-native run-android
```

## Usage
- Launch the app on a device or emulator.
- Create an account or log in.
- Browse the product catalog, add items to the cart, and proceed to checkout.
- Complete in-app purchases to simulate real transactions.
